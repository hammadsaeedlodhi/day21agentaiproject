# 🤖 Mini ReAct Agent (Groq + Web + Wikipedia + Arxiv + YouTube)

A lightweight **ReAct AI Agent** built with **Streamlit** and **Groq**.  
It follows the reasoning loop: **Thought → Action → Observation → Final Answer**  
and uses multiple tools to search real-world data.

---

## 🚀 Features
- 🧠 **ReAct Reasoning** — Thinks step-by-step like a real agent.
- 🌐 **Web Search** via DuckDuckGo.
- 📚 **Wikipedia Summaries.**
- 🧾 **Arxiv Research Papers.**
- 🎥 **YouTube Video Search.**
- ⚙️ Powered by **Groq** (ultra-fast LLMs).
